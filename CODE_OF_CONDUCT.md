# Code of conduct

## Our standard

Katalume contributors must communicate respectfully, review work on its merits,
protect user safety and privacy, and make the project welcoming regardless of
experience, identity, background, or location.

Unacceptable behavior includes harassment, discrimination, threats, sexualized
conduct, publishing private information, credential or vulnerability exposure,
and knowingly unsafe changes to code-execution infrastructure.

## Enforcement

Report conduct concerns privately to the GitHub organization owners until a
dedicated community contact is published. Owners should investigate promptly,
preserve confidentiality where possible, and apply proportionate corrective
action, including warnings, contribution restrictions, or removal.

This policy applies in repositories, reviews, issues, discussions, community
channels, and project events.
